# Layer Arch Orders

A payment processing API with layered architecture.

## Tech Stack

- Node.js + TypeScript
- Fastify (server)
- PostgreSQL (database)
- Redis (cache/queue)

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm

### Setup

```bash
pnpm install
```

### Development

```bash
pnpm run dev
```

Runs on `http://localhost:3333`
